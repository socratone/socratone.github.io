#### 고향사랑기부 서비스 프론트엔드 아키텍처 설계

고향사랑기부 서비스는 10만원 세액 공제와 답례품 혜택을 제공하는 신규 서비스로,
반응형 전환과 함께 레이아웃 구조 및 렌더링 전략 등 프론트엔드 아키텍처 핵심 영역 구현을 담당하며 서비스 출시를 이끌었습니다.

기존 모바일 고정 레이아웃 구조와 포탈로 분리된 navigation 제약으로 인해 레이아웃 단위 분리가 어려웠고,
이를 해결하기 위해 data-layout 속성에 글로벌 상태를 바인딩하여 동일한 마크업으로 다양한 레이아웃을 처리하는 구조를 설계했습니다.

또한 Next.js 기반으로 지역 상세 및 답례품 페이지에는 ISR을 적용해 성능과 데이터 최신성을 확보하고,
데이터 변경이 잦은 사용자 중심 페이지에는 PPR을 적용해 초기 로딩 속도를 개선했습니다.

- 고향사랑기부 25년 총 기부액 150억 달성

#### 고향사랑기부 홈페이지 성능 최적화

새로 출시된 고향사랑기부 홈페이지의 체감 속도가 느리다고 판단해 주도적으로 원인을 파악했고, HTML 문서 크기가 1MB를 넘는다는 점을 근거로 상급자에게 성능 개선을 제안했습니다.

HTML을 분석해보니 스와이퍼가 화면 밖 아이템까지 모두 렌더링하고 있어 보이지 않는 영역만큼 마크업이 중복 생성되고 있었고, 각 아이템에 포함된 복잡한 SVG가 HTML 텍스트 크기를 더욱 키우고 있었습니다.

SVG를 CDN 리소스로 분리해 최초 1회 요청 후 캐싱되도록 변경하여, HTML 텍스트 크기를 약 1/10 수준으로 줄였습니다.

- 라이트하우스 퍼포먼스 점수 +10, LCP -1초
